1.	Contacts Redux
    Контактная книга с разделами, избранными, группами и карточками контактов.  
    Проект переведён со стартового шаблона на архитектуру **Redux Toolkit**,  
    с сохранением структуры.

2.	Стек технологий
    - React 18 + TypeScript
    - Redux Toolkit (RTK, listener middleware, selectors)
    - React Router 6
    - Vite
    - Поддержка `alias`-импортов (`@/components`, `@/pages`, `@/redux` и т.д.)

3.	Основная идея
Проект демонстрирует переход от локального управления состоянием  
(через пропсы и useState) к централизованному Redux-хранилищу.

4.	Улучшения по сравнению со стартовым проектом
    - Переведено управление состоянием на “Redux Toolkit” (вместо локальных пропсов).
    - Реализованы отдельные “слайсы” для контактов, групп и избранного.
    - Добавлен “Redux ListenerMiddleware” для сохранения избранных контактов в “localStorage”.
    - Компоненты “ContactListPage”, “GroupPage”, “FavoritListPage” и “ContactPage” адаптированы под Redux-состояние.
    - Компонент “Breadcrumbs” синхронизирован с актуальными путями и Redux-данными.
    - Добавлен компонент “Layout” с общей структурой страницы и навигацией.
    - Нормализована структура импорта (`@/components`, `@/pages`, `@/redux` и т.д.).
    - Код отформатирован, структура файлов и именование приведены к единому стилю.

5.	Установка и запуск
    5.1.Установка зависимостей
    npm install
    5.2.Запуск в dev-режиме
    npm run dev
    5.3.Сборка production
    npm run build
